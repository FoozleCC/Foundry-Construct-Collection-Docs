# Getting Started

This documentation site is generated from the private Foundry source repository.

## Install addon files

1. Download the relevant `.c3addon` package.
2. Open Construct 3.
3. Install the addon in the Addon Manager.
4. Restart Construct 3 if prompted.

## Recommended first setup

1. Add the addon object/behavior to a test layout.
2. Configure editor properties.
3. Add basic event-sheet actions.
4. Verify runtime behavior in preview mode.

## Versioning notes

- Addons may version independently.
- Release notes summarize public behavior changes by addon.
