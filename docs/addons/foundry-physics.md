# Foundry Physics

## What it is

Foundry Physics is the world-level plugin for 3D physics integration.

## Core capabilities

- World lifecycle and stepping
- Query operations
- Runtime diagnostics

## Setup note

Create the Foundry Physics world plugin instance before adding physics body or
character behaviors, so those behaviors can register against an active world.
