# Addons

## Dependency order

1. Foundry Physics plugin (world owner)
2. Foundry Physics Body and/or Foundry Physics Character behaviors
3. Foundry Camera plugin
4. Foundry Camera Target behavior (optional, for weighted multi-target framing)
